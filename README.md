## 作成した機能について

本アプリでは、プレイヤーのレベルと敵のレベルを入力フォームに入力すると、
そのデータをもとに以下の内容を計算し、結果を出力します。

- 獲得できる報酬量
- 戦闘によって受けるダメージ量
- ダメージ回復に必要な時間

また、敵のレベル一覧を確認できるページも用意しており、
どのレベルの敵がワールドに出現しているかを一覧で確認できます。

レベル一覧は、報酬量や敵の攻撃力が変化する区切りごとに色分けしています。
これは、元データをExcelで管理していた際に、
数値の変化を視覚的に把握しやすくするために行っていた色分けを、
そのままWeb画面にも反映したものです。



## 工夫した点

- サイトを開いた際に、直感的に操作方法が分かる画面構成を意識しました。

- Excelで1つのセル内に関数として記述していた計算処理を、
  数式の意味から読み解き、Pythonで一つずつロジックとして書き起こしました。

- 敵の体力 ÷ 自身の攻撃力 で必要攻撃回数を計算した際、
  被ダメージ量 ＝ 必要攻撃回数 × 敵の攻撃力
  にしていたため、本来より1回多くダメージを受けてしまう問題に気付きました。
  そのため、計算時にごく小さな値（0.00001）を差し引くことで、
  実際の挙動に近い結果が出るよう調整しています。

- Excelでは1つのセル内に9つのIF文を入れて処理していたため、
  それらをPython側で条件ごとに分解して実装する点に苦労しました。

- 当初は、入力フォーム用と結果表示用でHTMLを分けて実装しようとしましたが、
  エラーが多発してしまったため、
  開発効率を優先し、1つのHTML内で完結する構成に変更しました。
